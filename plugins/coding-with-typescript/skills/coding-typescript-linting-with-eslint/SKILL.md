# Coding TypeScript Linting with ESLint

## Overview

ESLint with typescript-eslint is the standard linting solution for TypeScript projects. As of ESLint v9+ (and v10, released February 2026), flat config is the only supported configuration format. The legacy `.eslintrc.*` files are no longer honored in ESLint v10. typescript-eslint provides over 100 TypeScript-specific rules organized into tiered shared configs.
