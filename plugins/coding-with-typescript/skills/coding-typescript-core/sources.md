## Content Guidelines

- All code examples are original, written to illustrate patterns from the sources above
- Error codes (TS####) are verified against the TypeScript compiler source
- Version-specific features are tagged with their minimum TS version
- Runtime behavior (Node.js) is tested against the documented Node version
